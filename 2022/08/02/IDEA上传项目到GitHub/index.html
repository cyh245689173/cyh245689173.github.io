IDEA上传项目到GitHub

发表于 2022-08-02
踩坑日记

一、在github上创建仓库

点击头像左边的加号，创建新仓库

开始创建仓库

点击创建仓库

仓库创建完成：

二、在IDEA中上传项目代码到Github

1.首先测试自己的git是否安装成功

打开idea，File -> Setting->Version Control -> Git，或者直接搜索框搜索git

点击Test，测试git是否安装成功，弹出版本号即证明安装成功了。

2.在IDEA中添加GitHub

在Version Control中选择Github 点击右上方的加号，在弹出的界面中添加自己的github账号和密码

这里就添加成功了

如果产生了下图这个问题，可以参考这个博文：idea添加GitHub报错

3.上传项目到本地仓库

然后在idea上方依次点击 VCS -> Import into Version Control -> Create Git Repository

在弹出的界面中选择要提交的项目

然后右击项目选择 Git —> Add

这里可以看到正在添加

然后右击项目选择 Git -> Commit Directory，将项目提交到本地仓库

然后在弹出的信息输入自己的提交描述

然后点击commit进行提交，这样代码就提交到了本地仓库。

4.上传项目到远程仓库

然后就是将本地仓库的项目提交到远程仓库，Git -> Repository ->Push

在弹出的页面里点击Define remote

将我们的仓库地址粘贴进去

点击ok，开始push

下方会有push的进度条

等待push完成，在GitHub中就可以看到我们上传的项目啦。
