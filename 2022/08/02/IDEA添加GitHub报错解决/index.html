发表于 2022-08-02
踩坑日记

一、错误描述

使用IDEA添加github账号时出现以下错误：

二、错误解决，使用token登录

首先登录自己的GitHub

点击头像，打开Setting

在Setting中下滑，找到Developer settings，点击进入

依次点击Personal access tokens和Generate new token

然后note这里随便写一个

这里的权限全部选中

点击生成token

然后点击复制生成的token

然后点击Use Token

复制到Token里粘贴，点击登录

然后就登陆成功了
